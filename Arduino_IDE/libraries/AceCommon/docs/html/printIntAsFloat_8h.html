Detailed Description

Print integers as floating point numbers, without using floatin point operations which are expensive on 8-bit processors.

  • printUint16AsFloat3To()
    • divide an unsigned integer (uint16_t) by 1000 and print the result as a floating point number to 3 decimal places
  • printUint32AsFloat3To()
    • divide an unsigned integer (uint32_t) by 1000 and print the result as a floating point number to 3 decimal places
